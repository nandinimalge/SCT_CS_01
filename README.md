# Caesar Cipher

## About

This project is a Python implementation of the Caesar Cipher algorithm. It allows the user to encrypt and decrypt messages using a custom shift value.

## Features

* Encrypt a message
* Decrypt a message
* User-defined shift value
* Supports uppercase and lowercase letters
* Preserves spaces and special characters

## How to Run

Run the following command in the terminal:

python main.py

## Example

### Encryption

Message: Hello

Shift Value: 3

Encrypted Message: Khoor

### Decryption

Encrypted Message: Khoor

Shift Value: 3

Original Message: Hello
